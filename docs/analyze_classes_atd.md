# Основные АТД

Исходя из прошлого анализа, можно определить следующие АТД:
- **Player** (Игрок - изменяет состояние доски и компонентов на ней с помощью объединения в Комбинации)
- **Game** (Игра - сессия, за которую Игрок должен набрать необходимое количество очков; контролирует рамки начала и конца, содержит статистику и управляет бонусами)
- **Board** (Доска - игровое поле, которое содержит в себе Ячейки с Компонентами; ищет нужные комбинации, генерирует новые компоненты и заполняет ячейки; Доска размечена по координатам; очищает Ячейки, с помощью Бонусов или при нахождении Комбинаций)
- **Cell** (Ячейка - содержит в себе Компонент; может быть пустой, может быть заполненной, может удалять Компонент, имеет координату)
- **BoardComponent** (Компонент - фигурка обозначающая графический элемент на Доске; можно заменять фигурки на любой набор (цифры, эмодзи и тд))
- **Bonus** (Бонус - дополнительная возможность для Игрока, с помощью которой он может удалить с доски любую комбинацию из Компонентов (в зависимости от бонуса); у него есть характеристики, которые рангуют его и позволяют удалять разные комбинации Компонентов и зарабатывать больше очков; применяются пользователем)
- **Statistics** (Статистика - хранилище, в котором содержатся показатели Игрока, набранные за Игру - ходы, очки; аккумулирует и хранит показатели)
