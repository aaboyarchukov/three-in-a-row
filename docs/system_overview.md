# three-in-a-row overview

## Описание системы

В данном проекте нам необходимо реализовать игру "Три в ряд". Ее суть заключается в том, чтобы игрок собирал комбинации из 3 и более одинаковых объектов подряд на поле, путем переставления их. За каждую удаленную комбинацию игроку начисляются очки.

При нахождении таких комбинаций - они удаляются с игрового поля, затем генерируются новые объекты на поле - занимая освободившиеся места. Причем генерируются до тех пор, пока сбор комбинаций возможен.
С помощью удаления объектов с поля - игрок получает очки.

Также будет возможность собирать бонусы за более сложные комбинации. Бонусы будут игроку давать возможность удалять элементы в определенном порядке и в определенной форме.

По окончании очередной игры - игроку предоставляется статистика по его ходам и заработанным очкам.

## Сущности игры

**Игра** -- основная сущность проекта, которая управляет самой игрой, сохраняет статистику, историю, создает новую игру, инициализирует все зависимости (поле, игрока, объекты на поле)

**Поле** -- сущность, которая ответственна за управление полем с объектами: проверка комбинаций на поле, удаление объектов, генерация новых объектов и заполнение пустых мест.

Игрок -- действующее лицо игры, с помощью данной сущности мы можем управлять объектами на поле, пользоваться бонусами и заканчивать игру: переставление объектов на поле, использование доступных бонусов.

**Объект на поле** -- семантическая единица, обозначающая занятую клетку на поле.

**История игры** -- структура, в которой хранятся все ходы игрока в рамках одной игры.

**Статистика игрока** -- система, которая учитывает и хранит заработанные игроков очки, количество ходов, использованные бонусы.

**Бонусная система** -- система, которая отслеживает ходы игрока и автоматически выдает бонус при выстраивании сложной комбинации, бонус позволяет удалить объекты на поле различными способами (например, удаление всех элементов одного типа, удаление ряда или столбца)
